Mikro’s Lua API¶

The mik module wraps every functionality from Mikro’s C++ API to the Lua language.

Make sure you’ve read the Getting Started guide.

mik.quit():

Quits the game.

mik.btnp(name):

Returns true when a button has been pressed.

Parameters

name (string) – The button name. Button Constants.

mik.btnr(name):

Returns true when a button has been released.

Parameters

name (string) – The button name. Button Constants.

mik.btnh(name):

Returns true when a button has been held.

Parameters

name (string) – The button name. Button Constants.

mik.warp(x, y):

Warps the mouse to a specific location.

Parameters

  • x (number) – X coordinate.

  • y (number) – Y coordinate.

mik.mousex():

Gets the current mouse’s X coordinate.

mik.mousey():

Gets the current mouse’s Y coordinate.

mik.title([text]):

Gets/Sets the window title. If no argument is specified, it acts as a getter.

Parameters

text (string) – The text to set the window title to (optional).

Return type

string

Returns

The window title.

mik.key(r, g, b):

Sets the transparency key color.

Parameters

  • r (number) – Red component.

  • g (number) – Green component.

  • b (number) – Blue component.

mik.clip([x, y, w, h]):

Sets/Unsets the clipping region of the next drawing commands. If no argument is passed, or if any of the width and height components is zero, clipping will be disabled.

Parameters

  • x (number) – X component.

  • y (number) – Y component.

  • w (number) – Width component.

  • h (number) – Height component.

mik.clear([r, g, b]):

Clears the screen to an optional background color.

Parameters

  • r (number) – Red component.

  • g (number) – Green component.

  • b (number) – Blue component.

mik.dot(x, y, r, g, b):

Draws a dot (pixel) at a specified position (x, y) with a specified color (r, g, b).

Parameters

  • x (number) – X position component.

  • y (number) – Y position component.

  • r (number) – Red component.

  • g (number) – Green component.

  • b (number) – Blue component.

mik.line(x1, y1, x2, y2, r, g, b):

Draws a line between two point with a specified color (r, g, b).

Parameters

  • x1 (number) – Start X position component.

  • y1 (number) – Start Y position component.

  • x2 (number) – End X position component.

  • y2 (number) – End Y position component.

  • r (number) – Red component.

  • g (number) – Green component.

  • b (number) – Blue component.

mik.rect(x, y, w, h, r, g, b):

Draws a rectangle with a specified color (r, g, b).

Parameters

  • x (number) – X component.

  • y (number) – Y component.

  • w (number) – Width component.

  • h (number) – Height component.

  • r (number) – Red component.

  • g (number) – Green component.

  • b (number) – Blue component.

mik.rectf(x, y, w, h, r, g, b):

Draws a filled rectangle with a specified color (r, g, b).

Parameters

  • x (number) – X component.

  • y (number) – Y component.

  • w (number) – Width component.

  • h (number) – Height component.

  • r (number) – Red component.

  • g (number) – Green component.

  • b (number) – Blue component.

mik.circ(x, y, radius, r, g, b):

Draws a circle with a specified color (r, g, b).

Parameters

  • x (number) – X component.

  • y (number) – Y component.

  • radius (number) – Radius.

  • r (number) – Red component.

  • g (number) – Green component.

  • b (number) – Blue component.

mik.circf(x, y, radius, r, g, b):

Draws a filled circle with a specified color (r, g, b).

Parameters

  • x (number) – X component.

  • y (number) – Y component.

  • radius (number) – Radius.

  • r (number) – Red component.

  • g (number) – Green component.

  • b (number) – Blue component.

mik.spr(sprite, x, y, [sx, sy, sw, sh, flipx, flipy]):

Draws a sprite.

Parameters

  • sprite (Sprite object) – A sprite.

  • x (number) – X component.

  • y (number) – Y component.

  • sx (number) – Source rect X component (optional).

  • sy (number) – Source rect Y component (optional).

  • sw (number) – Source rect Width component (optional).

  • sh (number) – Source rect Height component (optional).

  • flipx (bool) – If true, flips the sprite horizontally (optional).

  • flipy (bool) – If true, flips the sprite vertically (optional).

mik.aspr(sprite, x, y, [flipx, flipy]):

Draws an animated sprite.

Parameters

  • sprite (Sprite object) – A sprite.

  • x (number) – X component.

  • y (number) – Y component.

  • flipx (bool) – If true, flips the sprite horizontally (optional).

  • flipy (bool) – If true, flips the sprite vertically (optional).

mik.tile(sprite, index, x, y, [flipx, flipy]):

Draws a sprite tile.

Parameters

  • sprite (Sprite object) – A sprite.

  • index (number) – Tile index.

  • x (number) – X component.

  • y (number) – Y component.

  • flipx (bool) – If true, flips the sprite horizontally (optional).

  • flipy (bool) – If true, flips the sprite vertically (optional).

mik.chr(font, map, char, x, y):

Draws a single character.

Parameters

  • font (Sprite object) – Font sprite.

  • map (string) – Character map string.

  • char (string) – Character to draw.

  • x (number) – X component.

  • y (number) – Y component.

mik.text(font, map, text, x, y):

Draws a string.

Parameters

  • font (Sprite object) – Font sprite.

  • map (string) – Character map string.

  • text (string) – Text.

  • x (number) – X component.

  • y (number) – Y component.

mik.load_sprite(fileName, [rows, cols]):

Loads a sprite from a PNG image file.

Parameters

  • fileName (string) – PNG image file name.

  • rows (int) – Number of vertical sprites in the sprite sheet (optional).

  • cols (int) – Number of horizontal sprites in the sprite sheet (optional).

Return type

Sprite object

Returns

A sprite.

mik.create_sprite(w, h):

Creates an empty sprite.

Parameters

  • w (number) – Sprite width.

  • h (number) – Sprite height.

Return type

Sprite object

Returns

A sprite.

mik.width([sprite]):

Gets the width of a sprite or the width of the screen if no parameter is specified.

Parameters

sprite (Sprite object) – A sprite.

mik.height([sprite]):

Gets the height of a sprite or the height of the screen if no parameter is specified.

Parameters

sprite (Sprite object) – A sprite.

mik.pget(sprite, x, y):

Gets a pixel at a certain position from a sprite.

Parameters

  • sprite (Sprite object) – A sprite.

  • x (number) – X component.

  • y (number) – Y component.

Return type

table {r, g, b}

Returns

The pixel color.

mik.pset(sprite, x, y, r, g, b, [ghost]):

Sets a pixel at a certain position in a sprite.

Parameters

  • sprite (Sprite object) – A sprite.

  • x (number) – X component.

  • y (number) – Y component.

  • r (number) – Red component.

  • g (number) – Green component.

  • b (number) – Blue component.

  • ghost (bool) – If true, the pixel will be marked as a “ghost pixel”, in other words, it will be ignored by the renderer, making it “transparent”.

mik.frame(sprite):

Gets the current frame from a sprite.

Parameters

sprite (Sprite object) – A sprite.

mik.animation(sprite):

Gets the current animation name from a sprite.

Parameters

sprite (Sprite object) – A sprite.

mik.play(sprite, name, speed, loop):

Plays a sprite animation.

Parameters

  • sprite (Sprite object) – A sprite.

  • name (string) – Animation name.

  • speed (number) – Animation speed in seconds.

  • loop (bool) – Whether or not the animation should loop.

mik.add_animation(sprite, name, [frames]):

Adds an animation to a sprite.

Parameters

  • sprite (Sprite object) – A sprite.

  • name (string) – Animation name.

  • frames (string) – Animation frames array (optional).

mik.load_sound(fileName):

Loads a sound from a WAV file.

Parameters

fileName (string) – WAV file name.

Return type

Sound object

Returns

A sound.

mik.play_sound(sound, [gain, pitch, pan]):

Plays a sound.

Parameters

  • sound (Sound object) – A sound.

  • gain (number) – The initial gain (volume) (optional).

  • pitch (number) – The initial pitch (optional).

  • pan (number) – The initial pan (optional).

Return type

Voice object

Returns

The voice this sound is going to be played in.

mik.play_music(sound, [gain, pan, fade]):

Plays a music.

Parameters

  • sound (Sound object) – A sound.

  • gain (number) – The initial gain (volume) (optional).

  • pan (number) – The initial pan (optional).

  • fade (number) – Fade time in seconds (optional).

mik.stop_music([fade]):

Stops the current music.

Parameters

fade (number) – Fade time in seconds (optional).

mik.volume(voice, [value]):

Gets/Sets the voice’s volume/gain. If no value argument is passed, it will act as a getter.

Parameters

  • voice (Voice object) – A voice from mik.play_sound.

  • value (number) – The volume/gain value [0.0 - 1.0].

mik.master_volume([value]):

Gets/Sets the master volume/gain. If no value argument is passed, it will act as a getter.

Parameters

value (number) – The volume/gain value [0.0 - 1.0].

mik.pitch(voice, [value]):

Gets/Sets the voice’s pitch. If no value argument is passed, it will act as a getter.

Parameters

  • voice (Voice object) – A voice from mik.play_sound.

  • value (number) – The pitch value [0.1 - 10.0].

mik.pan(voice, [value]):

Gets/Sets the voice’s pan. If no value argument is passed, it will act as a getter.

Parameters

  • voice (Voice object) – A voice from mik.play_sound.

  • value (number) – The pan value [-1.0 - 1.0].

Math Library Extensions¶

math.lerp(a, b, t):

Linear interpolation.

math.clamp(v, min, max):

Value clamping.

Button Constants¶

left¶

Left button. Mapped to Left Arrow Key and S Key.

right¶

Right button. Mapped to Right Arrow Key and F Key.

up¶

Up button. Mapped to Up Arrow Key and E Key.

down¶

Down button. Mapped to Down Arrow Key and D Key.

a¶

A button. Mapped to Z Key and K Key.

b¶

B button. Mapped to X Key and L Key.

start¶

Start button. Mapped to Enter Key and Numpad Enter Key.

left_mouse¶

Left mouse button.

middle_mouse¶

Middle mouse button.

right_mouse¶

Right mouse button.
